This is a cube animation that i made 
(Windows only).

Controls are 

P   -   Pause
O   -   Unpause
R   -   Refresh
Space   -   To quit

[works only in pause mode]
    WASD    -   for changing rotation in pause mode  
    JK,CV   -   for changing the camera and screen coordinates
    Arrow keys  -   to move around the cube
